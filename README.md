# 流萤主题数显表盘

流萤主题数显表盘是一款 Wear OS 表盘主题，提供灵活的自定义布局，适合追求个性化与简洁风格的用户。

## 主要特性
- 基于极坐标系进行布局设计，视觉风格独特且易于自定义
- 可自定义元素：时间、日期与电量的位置（极坐标）和文本大小
- 内置三张精美的Pixiv插画作为背景

## 使用说明
在设置各个元素的位置时，你需要输入两个数值：
- **方向**：0-359之间的整数，表示元素在表盘上的方向
- **距离**：0-100之间的整数，表示元素离表盘中心的远近（百分比）

设置界面会提供直观的输入框，你只需按照提示输入数值即可。

## 关于“极坐标系”的设计理念
本项目使用的定位方式其实很简单：
- **方向**：0° 指向表盘的12点钟方向，角度顺时针增大
  - 0° = 12点方向
  - 90° = 3点方向  
  - 180° = 6点方向
  - 270° = 9点方向
- **距离**：以表盘半径为单位，0%表示中心，100%表示边缘

这种定位方式特别适合圆形表盘，能精确控制每个元素的位置。

## 安装与使用
### 安装方法
1. 将APK安装包安装到Wear OS手表
2. 在手表上长按当前表盘，进入表盘选择界面
3. 找到“Firefly”表盘，点击应用

### 表盘设置
在手表上找到并打开“Firefly”应用图标，进入设置主界面：

#### 1. 元素配置（时间/日期/电量）
进入“元素配置”后选择要调整的元素：
- **方向设置**：进入方向选择界面后，可以直接**点击表盘预览图**上的任意位置定位，或使用底部的 **+ / - 按钮**进行1°微调
- **距离设置**：点击距离数值，输入0-100的整数（百分比）
- **大小设置**：点击大小数值，输入0-999的整数（如100表示1.00倍）
- **颜色设置**：点击颜色预览框，从预设颜色中选择，或选择“自定义”输入十六进制颜色值
- **重置功能**：**长按**底部的“重置”按钮可将该元素恢复为默认值

#### 2. 背景设置
- 进入后会列出所有背景图片（仅显示文件名）
- 当前选中的背景右侧会显示✓标记
- 点击任意背景文件名进入预览界面：
  - **单击预览图**可隐藏/显示底部的确认按钮
  - **长按预览图**可弹出缩放设置对话框，输入100-1000的整数百分比
  - 点击底部的圆形确认按钮保存选择

#### 3. 电量环开关
- 点击可开启/关闭表盘上的电量环显示
- 电量环终点带有动态火焰特效

### 设置生效说明
- 所有设置修改后会立即生效
- 表盘每秒钟刷新15次，保证流畅度的同时兼顾省电

## 常见问题
### Q1：如何快速定位元素到某个方向？
**A**：在方向选择界面，直接点击表盘预览图的任意位置，线段就会指向该方向。这是最快的方式！当然也可以用 +/- 按钮精确微调。

### Q2：背景图片缩放比例怎么理解？
**A**：缩放比例是相对于图片原始尺寸的百分比：
- 100%：原图大小，按中心裁剪填满屏幕
- 200%：放大到2倍，显示图片的中心区域（相当于裁掉了边缘）

系统会为每张背景图**独立保存**缩放设置，切换背景后不会互相影响。

### Q3：方表盘支持吗？
**A**：目前主要针对圆形表盘优化。方表盘上元素位置可能需要手动调整，避免靠近角落。另外，电量环也许不协调，可能需要关闭。

### Q4：为什么背景图显示不完整/被裁剪了？
**A**：你配置的缩放值大于100。表盘始终以**图片中心为基准**进行裁剪，确保填满整个屏幕。如果你希望显示更多区域，可以尝试降低缩放比例。

### Q5：我可以添加自己的背景图吗？
**A**：目前不支持添加。后续会考虑正式支持功能。背景图放在assets目录，你可以自行添加。
